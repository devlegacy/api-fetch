# Fetch - API 

## :package: How to install

```sh
  # Clone repo
  git clone https://github.com/devlegacy/fetch-api.git
```

## :rocket: How to use

```sh
  # Install local dependencies
  npm install && npm audit fix
  # Run project
  npm start
```
